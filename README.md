tudent Performance Analyzer

## Description
This is a Python-based command-line application that helps teachers manage and analyze student performance.

## Features
- Add student records
- View student records
- Update student performance
- Delete student records
- Automatic performance classification

## Technologies Used
- Python
- File handling

## How to Run
python student_analyzer.py
